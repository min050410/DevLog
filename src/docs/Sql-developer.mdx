정말 외우는 방법밖에 없는 것 같습니다.

문제에 서답형도 나오기 때문에 노트정리하며 용어를 유심히 보는 것을 추천드립니다.

이미 sql 문법에 익숙하신 분이라면 공부할 때 정말 도움이 됩니다.  
저같은 경우는 join 문법까지 배웠었는데 정말 많은 도움이 되었습니다.

이 부분을 공부하실 때 함수를 무작정 외우는 것보다 이론은 빠르게 빠르게 넘어가고
