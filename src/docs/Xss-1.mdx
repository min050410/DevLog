### 1. sql injection 공격이란?
sql injection은 sql문에 들어가야할 파라미터로 이상한 문자열을 삽입해 권한 등을 탈취할 수 있는 취약점입니다.

백엔드에 SQL문이 다음과 같이 작성되었다고 가정하겠습니다.
SELECT user FROM user_table WHERE id='입력한 아이디' AND password='입력한 비밀번호';
