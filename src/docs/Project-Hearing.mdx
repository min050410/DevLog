## 개요

청각 장애인을 위한 생활 도우미 서비스입니다.

## 담당한 기능

백엔드

- 유저
    - 유저 생성, 로그인
    - 내 정보 불러오기, 보호자 설정
- 알림
    - 내 알림 보기, 긴급 알림 POST
- 점수
    - 점수 등록, 내 모든 점수 확인, 오늘 점수 확인
- 대화
    - 대화 보내기
    - 같은 해시값 대화 받기
    - 오늘의 대화 횟수
    - 문장 마디 비율 확인
- 먼지, 조도 센서 값 받기

## 성과

2022 겨울방학 교내 해커톤 대회에서 우수상을 입상하였습니다. 그리고 교내 전시회에 출품하였습니다.

## 느낀점, 성능 개선점

- <big>swagger를 처음 도입해본 프로젝트 입니다.</big>  
swagger가 이렇게 편한지 몰랐고 앞으로도 이용해야겠다는 생각을 하였습니다.

- 백엔드에서 mqtt와 연동해 먼지 센서, 조도 센서를 불러오는데 성공하였습니다. 하지만 구현하고 나서 mqtt 같은 모듈들은 파이썬 백엔드 프레임워크로 구현하는 것이 더 효율적이라는 것을 알게 되었습니다.

- 방금 대화했던 내용을 같은 해시값으로 저장해서 내용을 전부 불러 올 수 있게 하였습니다. 해시를 사용하지 않는 것보다 조인을 하지 않아도 되어서 쿼리에서 성능 향상이 일어났습니다.

- 통계를 나타내는 부분이 백엔드에서 처리하기 까다로웠지만 리스폰스를 주고 나니 뿌듯했습니다.

- <big>해당 프로젝트를 통해 프론트의 입장을 많이 생각해 보게 되었습니다.</big>  
프론트가 값을 받고 어떻게 구현할 것인지 미리 생각을 하는 습관을 기르게 되었습니다.
