## 개요

삼류 웹소설 작가를 위한 소설 작가 전용 웹 에디터를 지원하는 일러스트, 만화, 소설 등을 올리고 공유하는 서비스입니다. 추가로 웹소설 작가들이 사용하기에 편리한 단축키 및 텍스트 효과를 지원하는 소설 작가 전용 웹 에디터를 제공합니다.

## 담당한 기능

백엔드

- 로그인, 회원가입 (구글 oauth)
- 프로필 조회
- 에디터
    - 자동 저장
    - 이미지 업로드
- 작품
    - 작품 업로드

## 느낀점, 성능 개선점
  
- <big>Kotiln을 이용한 개발</big>   
스프링을 사용할 때 항상 자바만 하다가 코틀린으로 개발을 하였습니다. 특히 코틀린으로 개발하면서 엔티티에서 신경을 쓸 부분이 많았고 kotiln 언어의 불변 지향성이 잘 드러나게 코드를 짜는 부분이 힘들었습니다.
