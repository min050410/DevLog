## 개요

농민분들이 시세를 확인하지 못하고 높은 가격에 파는 것이 안타까워서 개발을 시작하게 되었습니다.

OpenAPI를 통해서 농작물 실시간 가격을 가져오고 보여주는 앱입니다.

선형회귀모델을 이용하여서 내일 농작물 가격을 예측하는 기능도 추가되었습니다.

## 담당한 기능

백엔드, 프론트엔드

- 농산물 실시간 가격 가져오기
- 선형 회귀 AI를 통한 농산물 가격 예측하기
- 농산물 가격 검색기능

## 성과

2021 소프트웨이브에서 해당 작품을 출품하고 전시하였습니다.

## 느낀점, 성능 개선점

- AI를 올릴 때에는 백엔드 파이썬 라이브러리가 유리하다는 것을 알게 되었습니다. 당시에는 flask를 이용하여서 선형회귀 모델을 flask 서버에 올려서 요청을 응답하였습니다.
- 앱 개발을 토대로 react 개발을 시작하게 된 계기가 되었습니다.
