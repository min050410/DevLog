## 개요

BSMOJ는 백준 온라인 저지처럼 학교에서 알고리즘 문제를 푸는 프로젝트입니다.

자체적인 웹 기반 IDE가 포함되어 있어 제출한 뒤에도 코드 저장이 가능합니다.

## 담당한 기능

백엔드

- 코드 컴파일 및 실행
- 내 정보 불러오기 (제출 횟수, 맞, 틀, 컴파일 에러 횟수, 시간초과, 메모리 초과)
- 다른 사람 정보 불러오기
- 푼 문제들 모두 보기
- 회원가입/로그인
- 모든 문제 보기
- 내가 올린 문제 보기
- 문제 정보 보기(문제 제목, 문제 내용, 문제 난이도, 문제 출처, 시간제한, 메모리제한)
- 문제 올리기
- 문제 예제 올리기
- 문제 정답 테스트케이스 올리기
- 문제 삭제하기
- 문제 제출하기
- 대회 생성하기

## 성과

실제로 배포까지 진행되었지만 실제로 교내 알고리즘 대회에 사용되진 않았습니다.

## 느낀점, 성능 개선점

- 성능 개선을 위해 ORM없이 구현을 한 프로젝트입니다. ORM에 종속되지 않고 독립적으로 개발을 하였습니다. 직접 mariadb 모듈을 생성해서 nest 환경에서 연결하고 row query를 사용하였습니다.

- 프론트와 CORS문제에 직면하고 힘들게 해결한 프로젝트입니다. 쿠키 관련해서 이슈가 터졌고 프론트 쪽에서 withCredentials 헤더를 포함하여서 해결하였습니다.

- 배포는 완료 되었지만 실제로 실행되지 않은 것이 아쉽습니다. 다음에는 프로젝트를 만들고 더 홍보를 하거나 선생님께 연락하는 식으로 프로젝트를 실행할 계획부터 짜는 것이 중요하다는 것을 알게 되었습니다.
