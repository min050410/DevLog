## 개요

잘 알려진 2048 게임을 React로 구현해 실력을 키우고 싶어서

간단한 웹에서 동작하는 2048 게임을 제작하였습니다.

## 담당한 기능

백엔드, 프론트엔드

- 2048 게임 로직 구현 + 점수
- 랭킹 기능 구현
- 내 정보 불러오기
- 유저 이름 변경

## 성과

교내 웹 게임 대회에서 2등 상을 받았습니다.
